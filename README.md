# Search-Engine-with-Langchain-Tools-and-agents


An interactive Streamlit chatbot powered by LangChain Agents and Groq LLaMA-3, capable of searching the web, querying Wikipedia, and fetching research papers from arXiv using natural language.

This app demonstrates how an AI agent reasons step-by-step, selects the right tool, and responds with accurate, up-to-date information.
